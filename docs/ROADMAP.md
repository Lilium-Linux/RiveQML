# Roadmap

## Next Priorities

1. Add more automated coverage for rendering and interaction, not just file loading.
2. Tighten install-tree validation with a dedicated consumer-package test.
3. Add a cleaner public story for overriding external asset roots on shared `RiveFile` instances.
4. Add runtime-to-QML view-model synchronization for full two-way binding.
5. Introduce native GPU backends after the package and behavior surface are stable.

## Longer Term

- broader renderer backend support beyond the current macOS CoreGraphics path
- GPU-native Qt Quick integration
- richer runtime event surfacing into QML
- stronger performance profiling and frame-time telemetry
